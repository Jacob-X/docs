## 第一个Python程序
使用记事本创建一个后缀名为`py`的文件，如`test.py`，将下面的内容复制进去，并保存在`D:/test.py`下。

```python{2}
# -*- coding: utf-8 -*-
print('Hello, 塔容万物')
```
使用`Windows` :window: + `R`键，打开命令行程序，输入下面的命令，并回车运行
```bash
python D:/test.py
```

🎉命令行上会显示
```bash
Hello, 塔容万物
```
